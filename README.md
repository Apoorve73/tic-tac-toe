### To run this program offline
using terminal or command prompt 
- Fork this [repo](https://github.com/swaaz/tic-tac-toe)
- Clone the repo
```
git clone [url]
```
#### replace [ url ] with the link copied from clone option
- Change Directory
```
cd tic-tac-toe
```
- To run program
```
python3 program.py
```
